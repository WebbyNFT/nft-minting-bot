### 1. Installing dependencies
To install all dependencies run:
```sh 
npm install
```
### 2.Configuring:
To configure copy  **.env.example** file, change urls and values there and save as **.env** file.

### 3.Minting
To mint just simply run 

```sh
npm start
```
